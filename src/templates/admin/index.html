ABUAD HMS Admin Application

The ABUAD HMS Admin Application is an innovative tool that enables administrators to manage students and lecturers in the hostel monitoring system of Afe Babalola University, Ado-Ekiti (ABUAD). The system provides a range of features to support the efficient management of the university community.

Create and Update Students and Lecturers

The ABUAD HMS Admin Application allows administrators to create new student and lecturer profiles, as well as update existing ones with new information as needed.

Read and View Student and Lecturer Profiles

The ABUAD HMS Admin Application enables administrators to access and view the profiles of all students and lecturers in the hostel monitoring system, providing a comprehensive overview of the university community.

Delete Student and Lecturer Profiles

The ABUAD HMS Admin Application allows administrators to delete student and lecturer profiles as needed, ensuring that the system is up-to-date and accurate at all times.
